# About Me — Advait Anil

A simple personal webpage built as part of the ThinkPlan Pre-Internship Training Programme.

## What's in it

- **index.html** — the main page with a bio, skills list, contact section, and a greeting button
- **style.css** — styling for fonts, colours, layout, and spacing
- **script.js** — JavaScript with three functions (greet, add, isEven) and a button handler

## How to view it

Just open `index.html` in your browser. No build step needed.

## Built with

- HTML
- CSS
- Vanilla JavaScript
